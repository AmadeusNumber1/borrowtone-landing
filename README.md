# BorrowTone Landing Page

Landing page concept for an "Airbnb for instruments" platform.

## What was done

- Market/UX research on marketplace landing page conversion and rental trust patterns
- Sleek, responsive one-page design
- Custom generated SVG visuals (hero + 3 instrument cards)
- Sections included:
  - Hero with clear value prop + CTAs
  - Featured inventory cards
  - How it works (3-step flow)
  - Trust/safety and deposit messaging

## Run locally

```bash
cd instrumentbnb
python3 -m http.server 4173
# open http://127.0.0.1:4173
```

## Files

- `index.html`
- `styles.css`
- `assets/*.svg`

## Notes from research

1. Trust-first UX matters in peer-to-peer rentals (verification, reviews, condition checks)
2. Explicit deposit and insurance options reduce lender friction
3. Landing pages convert better with clear above-the-fold value and immediate CTA options
4. Keep social proof and safety details visible early
